# Console Administration

The **NetBox Labs Console** ([console.netboxlabs.com](https://console.netboxlabs.com)) is the central hub for managing your NetBox Cloud organisation and instances.

## What You Can Do in the Console

| Task | Location |
|------|----------|
| Create/delete instances | **Instances** tab |
| Manage org members | **Organization → Members** |
| Configure SSO | **Organization → Security** |
| View billing and usage | **Organization → Billing** |
| Generate API tokens | **Account → API Tokens** |
| Access support tickets | **Help → Support** |

## Console Sections

### Instances
List, create, start, stop, and delete NetBox instances. Each instance has its own direct URL (e.g. `https://yourorg.cloud.netboxlabs.com`).

### Organization
Manage members, roles, SSO providers, and billing details for your organisation.

### Account
Personal settings — email, password, MFA, and personal API tokens.
