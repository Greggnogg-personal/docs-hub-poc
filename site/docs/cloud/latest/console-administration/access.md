# Accessing the Console

## Sign In

1. Go to [console.netboxlabs.com](https://console.netboxlabs.com)
2. Enter your email and password, or use your SSO provider if configured
3. Complete MFA if enabled

## First-Time Login

After signing up you'll receive a verification email. Click the link to activate your account, then log in.

## MFA

We strongly recommend enabling multi-factor authentication:

1. Go to **Account → Security**
2. Click **Enable MFA**
3. Scan the QR code with your authenticator app
4. Enter the 6-digit code to verify

## SSO

If your organisation uses SSO (Okta, Azure AD, Google Workspace), an administrator must configure it in **Organization → Security** before users can sign in via SSO.
