# NetBox Cloud Documentation

NetBox Cloud is the fully-managed SaaS version of NetBox. Zero infrastructure maintenance — always up to date, enterprise-grade, globally available.

## Quick Start

| Step | Action |
|------|--------|
| 1 | [Sign up](https://cloud.netboxlabs.com/signup) for a free account |
| 2 | [Access the Console](./console-administration/overview) |
| 3 | [Create your first instance](./instance-management/create) |
| 4 | Log in to NetBox and start documenting your infrastructure |

## Key Features

- **Zero Infrastructure** — no servers, no patches, no scaling concerns
- **Automatic Updates** — always running the latest stable NetBox
- **SOC 2 Type II** — enterprise-grade security controls
- **99.9% SLA** — multi-region redundancy
- **Expert Support** — from the team that builds NetBox
