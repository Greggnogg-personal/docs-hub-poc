# Create a NetBox Cloud Instance

## Prerequisites

- An active NetBox Cloud account
- **Admin** or **Owner** role in your organisation

## Steps

1. Log in to the [Console](https://console.netboxlabs.com)
2. Click **Instances → New Instance**
3. Choose a plan (Free, Professional, or Enterprise)
4. Select a **Region** (US East, EU West, AP Southeast)
5. Enter an **Instance Name** — this becomes part of the URL:  
   `https://<name>.<org>.cloud.netboxlabs.com`
6. Click **Create**

Instance provisioning takes approximately 2–3 minutes.

## After Creation

- You'll receive an email confirmation with the instance URL
- The first login uses your Console credentials
- Set a memorable admin password inside NetBox under **Admin → My Profile**

## Free Plan Limits

Free instances include:
- Up to **1,000 objects**
- Up to **5 users**
- Community support only

Upgrade at any time in **Organization → Billing**.
