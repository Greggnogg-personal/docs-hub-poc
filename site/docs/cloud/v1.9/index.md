- Basic cloud access and administration
- Core backup and restore functionality  
- Instance management and monitoring
- Basic user management and permissions
- Legacy authentication methods
- Simple configuration management

:::info About v1.9 Cloud
Cloud v1.9 provided basic cloud instance management capabilities with a simplified interface. Key features included:

- Instance start/stop/restart controls
- Basic monitoring and status information
- Manual backup creation and management
- Simple user account administration
- Access to instance credentials and API tokens
- Basic configuration management
:::

## Migrating to v1.10+

The v1.10+ releases have been reorganized into a more integrated structure. See the current [cloud administration documentation](/docs/cloud/console-administration/console-overview) for the latest features and improvements.
