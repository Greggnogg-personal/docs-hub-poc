# Enterprise Feature Overview

NetBox Enterprise builds on the Community edition with production-hardening and organisational features.

## SSO & Identity

Integrate with any SAML 2.0 or OIDC-compatible identity provider (Okta, Azure AD, Ping Identity, Google Workspace). See [SSO & SAML](./sso).

## High Availability

Active/passive HA with automatic failover. PostgreSQL streaming replication and Redis Sentinel included. See [High Availability](./ha).

## Audit Log

Every create, update, and delete action is recorded with the user, timestamp, and changed fields. Exportable to SIEM systems via syslog or webhook. See [Audit Log](./audit-log).

## Role-Based Access Control

Granular object-level permissions go beyond Community's group-based model, allowing per-object-type read/write/delete control per user or group.

## Enterprise Support

- **SLA**: 4-hour response for P1 issues
- **Channels**: email, Slack Connect, video call
- **Named CSM**: dedicated Customer Success Manager (Enterprise tier)
