# NetBox Community Edition

Welcome to the NetBox Community Edition documentation — the open-source network source of truth trusted by thousands of organizations worldwide.

## What's Included

The Community Edition provides full access to the core NetBox platform and its ecosystem:

- **[NetBox](../netbox/index.md)** — The open-source network source of truth for modeling and documenting network infrastructure
- **[Diode](../diode/index.md)** — Data ingestion service for pushing network state into NetBox from automated sources
- **[Copilot](../copilot/index.md)** — AI-powered assistant for querying and managing your NetBox data
- **[MCP Server](../mcp/index.md)** — Model Context Protocol server for AI agent integration with NetBox
- **[Integrations](../integrations/tool-integrations/netbox-ansible-collection.md)** — Ansible and pyATS integrations for network automation
- **[Extensions](../extensions/branching/index.md)** — Branching for change management workflows

## Getting Started

If you're new to NetBox, start with the [NetBox documentation](../netbox/index.md) to understand the core platform, then explore the additional tools and integrations available in the Community Edition.

## Upgrading to Enterprise or Cloud

Looking for additional features like vendor integrations, SSO/LDAP authentication, simplified deployment, or managed infrastructure?

- **[NetBox Enterprise](../enterprise/enterprise-features/nbe-overview.md)** — Self-hosted distribution with enterprise features and professional support
- **[NetBox Cloud](../cloud/index.md)** — Fully-managed SaaS version eliminating infrastructure complexity
