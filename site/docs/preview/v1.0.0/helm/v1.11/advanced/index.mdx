## When to Use Advanced Scenarios

Use advanced deployment scenarios when you need:

- **Private registry integration** for air-gapped or security-restricted environments
- **Automated CI/CD pipelines** for continuous deployment and infrastructure as code
- **Complex networking setups** with custom ingress, load balancer, or service mesh configurations
- **High availability deployments** with external databases, redundancy, and disaster recovery
- **Security hardening** with custom RBAC, network policies, and compliance requirements

## Getting Started

1. **Start with platform-specific guides** - Choose your deployment platform and follow the complete setup
2. **Follow automation scripts** - Use provided scripts for consistent and repeatable deployments
3. **Customize for your environment** - Adapt examples to your specific infrastructure and requirements
4. **Test in non-production first** - Validate all configurations before production deployment
5. **Plan for operations** - Ensure monitoring, backup, and maintenance procedures are in place
